# 写作风格（强制遵守）

## 语言特点
- 口语化、短句、一句一段
- 自黑、接地气、不装
- 固定开头："大家好，我是老里"（50-100字内切入）
- 固定结尾：感恩三连（不可省略）

## 真实性要求
- ✅ 要有真实的细节（谈价格、砍价、看场地）
- ✅ 要有真实的情绪（犹豫、纠结、绝望）
- ❌ 避免太完美的故事、太戏剧化的转折
- ❌ 避免太正能量的结尾

## 禁止
- AI味句式、书面腔、说教口吻
- 被动句、冷冰冰总结
- 配图文字用英文（必须用中文）

## 结构公式
故事 → 冲突 → 框架 → 升华

# 工作流程

## 写文章流程
1. 检索灵感库找素材
2. 生成3个不同角度的大纲
3. 等用户确认后生成初稿
4. 去AI味（检查真实性）
5. 生成配图（3-5张）
6. 排版（Raphael Claude 燕麦色主题）
7. 自动插入配图到HTML
8. 发布到公众号草稿

## 配图要求
- 文字必须是中文
- 风格：故事性插图 > 信息图
- 情绪类文章用故事性场景图
- 技术类文章用信息图
- 每篇3-5张

## 排版要求
- 必须用 Raphael Claude 燕麦色主题
- 特点：暖黄底色、砖红标题、淡背景高亮

# 常见任务

## 生成文章
```
1. 检索素材库
2. 生成大纲（3个角度）
3. 等用户确认
4. 生成初稿
5. 去AI味
6. 配图 + 排版 + 发布
```

## 配图流程
```bash
# 1. 生成图片（使用通义万相或其他工具）
# 2. 上传到微信素材库
curl -F "media=@图片.png" \
  "https://api.weixin.qq.com/cgi-bin/material/add_material?access_token=${TOKEN}&type=image"

# 3. 获取URL后插入HTML
# 在合适位置插入：<img src="URL" style="max-width:100%;">
```

## 发布流程
```bash
# 1. 获取 access_token
curl "https://api.weixin.qq.com/cgi-bin/token?grant_type=client_credential&appid=AppID&secret=Secret"

# 2. 上传封面图（获取 media_id）
# 3. 创建草稿（包含HTML内容和封面）
```
